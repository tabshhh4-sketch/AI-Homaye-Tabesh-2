# همای تابش (Homaye Tabesh)

افزونه‌ای میانلایه، ماژولار و هدایتگر کاربران، قدرتگرفته از هوش مصنوعی

## نمای کلی

همای تابش یک افزونه وردپرس است که بهعنوان هاب هوشمند برای هماهنگی، تصمیم‌گیری و راهنمایی تمام فرآیندهای کاربران وبسایت عمل می‌کند.

## الزامات فنی

- **PHP:** نسخه 8.2 یا بالاتر
- **WordPress:** نسخه 6.0 یا بالاتر
- **Node.js:** نسخه 18 یا بالاتر (برای توسعه)
- **Composer:** برای مدیریت وابستگی‌های PHP

## معماری

این افزونه با معماری ماژولار طراحی شده است که امکان توسعه، نگهداری و افزودن قابلیت‌های جدید را به‌صورت مستقل فراهم می‌کند.

### ساختار دایرکتوری

```
homaye-tabesh/
├── assets/                    # فایل‌های استاتیک
│   ├── css/                   # استایل‌های CSS
│   ├── js/                    # کدهای JavaScript/React
│   │   └── src/              # سورس React
│   └── dist/                  # فایل‌های بیلد شده
├── includes/                  # کلاس‌های PHP
│   ├── Core/                  # هسته اصلی افزونه
│   │   ├── Plugin.php        # کلاس اصلی (Singleton)
│   │   ├── Activator.php     # مدیریت فعال‌سازی
│   │   ├── Deactivator.php   # مدیریت غیرفعال‌سازی
│   │   ├── MenuManager.php   # مدیریت منوها
│   │   └── AdminInterface.php # رابط مدیریتی
│   └── Admin/                 # بخش‌های مدیریتی
│       ├── AtlasControlCenter/    # مرکز کنترل اطلس
│       └── HomaSuperPanel/        # سوپرپنل هما
├── composer.json              # وابستگی‌های PHP
├── package.json               # وابستگی‌های NPM
└── homaye-tabesh.php         # فایل اصلی افزونه
```

## نصب و راه‌اندازی

### نصب وابستگی‌ها

```bash
# نصب وابستگی‌های PHP
composer install --no-dev --optimize-autoloader

# نصب وابستگی‌های NPM
npm install --legacy-peer-deps

# بیلد کردن React
npm run build
```

### فعال‌سازی در وردپرس

1. فایل‌های افزونه را در دایرکتوری `/wp-content/plugins/homaye-tabesh/` قرار دهید
2. از پنل مدیریت وردپرس، به بخش Plugins بروید
3. افزونه "همای تابش" را فعال کنید

## منوهای مدیریتی

### 1. مرکز کنترل اطلس

منوی اصلی برای نظارت و کنترل سیستم با زیرمنوهای زیر:

- **نمای کلان:** نمای کلی از وضعیت سیستم
- **مداخله زنده:** مداخله و کنترل زنده
- **تحلیل رفتار:** تحلیل رفتار کاربران
- **پیشنهادات:** پیشنهادات هوشمند
- **شبیه‌ساز تصمیم:** شبیه‌سازی تصمیمات
- **تنظیمات هسته:** تنظیمات اصلی سیستم
- **پیکربندی پیشرفته:** تنظیمات پیشرفته

### 2. سوپرپنل هما

منوی اصلی برای مدیریت کلی با زیرمنوهای زیر:

- **داشبورد اجرایی:** داشبورد مدیریتی
- **مدیریت کاربران:** مدیریت کاربران
- **سلامت و عیب‌یابی:** بررسی سلامت سیستم
- **توسعه مغز:** توسعه قابلیت‌های هوش مصنوعی
- **امنیت:** مدیریت امنیت
- **ناظر کل:** نظارت جامع
- **پرسونا:** مدیریت شخصیت هوش مصنوعی

## ویژگی‌های فنی

### PSR-4 Autoloading

از استاندارد PSR-4 برای بارگذاری خودکار کلاس‌ها استفاده می‌شود:

```php
"autoload": {
    "psr-4": {
        "HomayeTabesh\\": "includes/"
    }
}
```

### Strict Types

تمام فایل‌های PHP از `declare(strict_types=1)` استفاده می‌کنند برای امنیت و کارایی بالاتر.

### معماری Singleton

کلاس اصلی Plugin از الگوی Singleton استفاده می‌کند تا تنها یک نمونه از افزونه در حافظه وجود داشته باشد.

### رابط کاربری React

رابط مدیریتی با React پیاده‌سازی شده است و از WordPress REST API برای تبادل داده استفاده می‌کند.

## توسعه

### اجرای محیط توسعه

```bash
# Watch mode برای React
npm run dev
```

### ساختار کامپوننت‌های React

```
assets/js/src/
├── index.js                      # نقطه ورود
├── App.js                        # کامپوننت اصلی
├── styles.css                    # استایل‌ها
└── components/
    ├── AtlasControlCenter.js     # کامپوننت مرکز کنترل اطلس
    └── HomaSuperPanel.js         # کامپوننت سوپرپنل هما
```

- استفاده از WordPress nonces برای امنیت AJAX
- بررسی دسترسی‌ها (`manage_options`)
